Simple, clean website to access 7 free AI models for text generation.

## What's Included

- **7 Free AI Models**: GPT-4o, GPT-4 Turbo, Gemini Pro, Command R+, Mixtral 8x7B, Mistral Large, Llama 2 70B
- **Simple Interface**: No bloat, just what you need
- **API Key Storage**: Keys saved locally in your browser only
- **Mobile Friendly**: Works on desktop and mobile
