## Troubleshooting

### Models not showing in dropdown?
- Hard refresh: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
- Check if models list appears at bottom of page

### API Settings button not opening?
- Clear browser cache
- Try opening in incognito/private window

### Getting API errors when sending a prompt?
1. Check that you copied the API key correctly
2. Make sure you clicked "Save Keys" button
3. Verify the API key isn't expired
4. Try a different model/provider

### Keys not saving?
- Make sure you clicked the "üíæ Save Keys" button
- Check browser's localStorage is enabled
- Try in a private/incognito window
